# Staff Timetable

スタッフのシフト表を**閲覧・編集**できる、シンプルな静的 Web アプリです。
ビルド不要の素の HTML / CSS / JavaScript で作られており、**GitHub Pages** にそのまま公開できます。

## 主な機能

- 週単位のシフト表表示（月曜始まり、土日を色分け）
- 前週／次週／今週への移動
- 編集モードの切り替え
  - スタッフの追加・名前編集・削除
  - セルをクリックしてシフト入力（プリセット or 自由入力）
- シフトプリセット（早番・遅番・日勤・夜勤・休み・有休）と凡例表示
- データはブラウザの `localStorage` に自動保存
- JSON でのエクスポート／インポート（バックアップ・共有用）
- 印刷対応（ツールバー等を除いた表のみ印刷）

## ローカルでの確認

`index.html` をブラウザで開くだけで動作します。

```sh
# 簡易サーバーで開く場合（任意）
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## GitHub Pages への公開手順

1. このリポジトリを GitHub に push します。
2. GitHub のリポジトリページで **Settings → Pages** を開きます。
3. **Build and deployment** の **Source** を `Deploy from a branch` に設定。
4. Branch を `main` / フォルダを `/ (root)` に設定して **Save**。
5. 数分後、`https://<ユーザー名>.github.io/Staff_timetable/` で公開されます。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | 画面構造 |
| `style.css` | スタイル |
| `app.js` | シフト表のロジック・データ保存 |

## データについて

シフトデータは**閲覧している端末のブラウザ内**にのみ保存されます。
別の端末や他の人と共有したい場合は、メニューの「JSON エクスポート」で書き出し、
相手側で「JSON インポート」してください。
